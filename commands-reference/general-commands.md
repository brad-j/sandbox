## Useful Linux Commands
- top + htop (processes)
- ps + ps aux (processes)
- df + df -h (diskfree)
- ls -lh
- chmod, chown
- systemctl (system control)
- journalctl -u [**service**] (-f for follow mode)
- rsync -avz (--dry-run)
- ln -sf (symbolic link with force flag)
